A Fissure in the Community

Mozilla and Opera present a competing vision:

  1. Backwards compatibility with clear migration path
  2. Well-defined error handling
  3. User should not be exposed to authoring mistakes
  4. Practical use
  5. Scripting is here to stay
  6. Device-specific profiling should be avoided
  7. Open processes

W3C refused to devote resources
Web Hypertext Application Technology WG was born
